Section 01
Where Every Answer Comes From
ℹ️ This chatbot uses RAG (Retrieval-Augmented Generation) — answers are always grounded in real Liberty Dental content, never invented. The LLM cannot answer from memory; it can only use what Pinecone retrieves.
INDEXING PIPELINE (runs once via data_pipeline.py)
📝
Contentful CMS
All website pages, FAQs, and content blocks from the UAT environment
⚙️
data_pipeline.py
Extracts text fields + rich text, adds 8 seed knowledge items
🔢
Embeddings
all-MiniLM-L6-v2 converts each doc to a 384-dim vector
🗄️
Pinecone Index
"liberty-dental-kb" stores all vectors + metadata (title, content)
QUERY PIPELINE (runs on every user message)
💬
User Question
Text typed into the chat input
🔢
Embed Query
Same model turns the question into a 384-dim vector
🔍
Pinecone Search
Finds top 6 most similar docs. Score must be ≥ 0.45
🦙
Llama 3.3 70B
Generates a natural-language answer from retrieved context only
🔗
Answer + Links
Response + relevant links shown to the user
💡 Fallback LLM: If Llama/Groq is unavailable, Gemini 2.5 Flash takes over automatically. The user never sees an error — they just get an answer.
Section 02
The 3 Routing Layers — How a Question Is Decided
L1
🚫 Restricted Pattern Check (hardcoded, no AI)
If the question contains sensitive personal-data terms, the bot immediately blocks it and redirects to the Member Portal. Pinecone and Llama are never called.
Triggers on: "member id", "SSN", "date of birth", "my claim", "my eligibility", "password", "medical advice", "diagnose"
L2
💬 Capabilities Check (hardcoded, no AI)
If the question is "what can you do?"-style, the bot returns a fixed list of everything it can help with. Pinecone and Llama are not called.
Triggers on: "how can you help", "what can I ask", "what can I search", "who are you", "what do you do", "how does this work"
L3
🔍 Semantic Search → Llama 3.3 70B (AI-generated)
All other questions embed the query, search Pinecone for the 6 most relevant docs (score ≥ 0.45), then Llama generates an answer from that context only.
Everything else: dental plan questions, finding a dentist, app download, teledentistry, broker/provider questions, etc.
Section 03
Keyword Search or Semantic Search?
Not used
Keyword Search
Matches exact words. If you ask "dental coverage" and the doc says "dental benefits", it finds nothing. Fails on synonyms, abbreviations, and differently-worded questions.
Ask: "tooth pain emergency" → Finds: nothing (no exact match)
✓ Used here
Semantic Search via Pinecone
Understands meaning, not words. The question and every document are converted into vectors — similar meanings land close together. Handles paraphrases, typos, and informal language.
Ask: "tooth pain emergency" → Finds: Teledentistry & Emergencies doc ✓
ℹ️ Score threshold is 0.45. If the best Pinecone match scores below 0.45 (out of 1.0), the bot says it cannot find the answer. This prevents low-quality or off-topic answers from getting through. Links are only suggested when the answer text contains trigger words (see next section).
Section 04
All 13 Links — What Triggers Each One
💡 Links are suggested by keyword scanning the LLM's answer text + the user's question (not semantic search). If the word "app" or "download" appears in either, both app store links are added. This is why not every answer gets links — only when trigger words appear.
Link Name Status How It Gets Triggered
Find a Dentist ✓ Works keyword answer/question contains "find" AND "dentist"
Member Login ✓ Works keyword answer/question contains "login" or "portal"
Mobile App (Apple) ✓ Works keyword answer/question contains "app" or "download"
Mobile App (Android) ✓ Works keyword same as Apple — both always show together
Teledentistry ✓ Works keyword answer/question contains "teledentistry" or "emergency"
Shop Plans ✓ Works keyword answer/question contains "shop", "buy", or "individual"
Join Provider Network ✓ Works keyword answer/question contains "provider" or "join"
Brokers ✓ Works keyword answer/question contains "broker"
Broker Contact ✓ Works keyword same as Brokers — both show together
File a Grievance or Appeal ✓ Works keyword answer/question contains "grievance" or "appeal"
Contact Liberty ✓ Works Hardcoded fallback — always shown when no Pinecone match found
Members ✓ Works Hardcoded — shown in Capabilities response only
Home ⚠ Never shown In the PUBLIC_LINKS dictionary but no code ever adds it to suggestions
